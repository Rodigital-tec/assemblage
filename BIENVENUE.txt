ETAPES PRINCIPALES
════════════════════════════════════════════════════════════════════════════

Voici les 5 étapes principales :

1️⃣  Initialiser Git
    $ git init
    $ git add .
    $ git commit -m "Préparation déploiement"

2️⃣  Créer un dépôt GitHub
    → https://github.com/new
    → Créez un dépôt "assemblage"

3️⃣  Pousser votre code
    $ git remote add origin https://github.com/VOTRE_USERNAME/assemblage.git
    $ git branch -M main
    $ git push -u origin main

4️⃣  Déployer sur Render
    → https://render.com
    → Créez un "Web Service"
    → Sélectionnez votre dépôt

5️⃣  Configurer les variables
    → FLASK_ENV=production
    → SECRET_KEY=your_secret_key
