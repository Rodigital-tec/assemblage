ETAPE 1: LANCER LE SCRIPT (5 minutes)
─────────────────────────────────────

Windows:
────────
Double-cliquez sur: deployer.bat

OU ouvrez le terminal et tapez:
deployer.bat


Mac/Linux:
──────────
Ouvrez le terminal et tapez:
bash deployer.sh


ETAPE 2: ENTRER VOTRE NOM D'UTILISATEUR GITHUB
───────────────────────────────────────────────

Le script vous demandera votre nom d'utilisateur GitHub.

Tapez-le et appuyez sur Entrée.

Le script fera le reste automatiquement ! ✅


ETAPE 3: DEPLOYER SUR RENDER (5 minutes)
─────────────────────────────────────────

Le script affichera les instructions pour Render.

Suivez-les simplement:

1. Allez sur: https://render.com
2. Connectez-vous avec GitHub
3. Cliquez sur "New +" puis "Web Service"
4. Sélectionnez votre dépôt "assemblage"
5. Remplissez les champs et cliquez sur "Create Web Service"
6. Attendez 2-3 minutes

════════════════════════════════════════════════════════════════════════════
