ETAPE 2 : AVOIR LES COMPTES NECESSAIRES
════════════════════════════════════════════════════════════════════════════

Vous avez besoin de :

1. Un compte GitHub (gratuit)
   → Allez sur https://github.com
   → Cliquez "Sign up"
   → Créez un compte

2. Un compte Render (gratuit)
   → Allez sur https://render.com
   → Cliquez "Sign up"
   → Connectez-vous avec GitHub


ETAPE 3 : INITIALISER GIT
════════════════════════════════════════════════════════════════════════════

Ouvrez un terminal/PowerShell dans votre dossier et exécutez :

$ git init
$ git add .
$ git commit -m "Préparation déploiement"

Vous devriez voir quelque chose comme :
✓ Initialized empty Git repository
✓ create mode 100644 Procfile
✓ ...


ETAPE 4 : CREER UN DEPOT GITHUB
════════════════════════════════════════════════════════════════════════════

1. Allez sur https://github.com/new

2. Remplissez :
   - Repository name: assemblage
   - Description: Application Assembl'âge
   - Sélectionnez "Public"

3. Cliquez "Create repository"

4. Vous verrez une page avec des instructions. Copiez l'URL :
   https://github.com/VOTRE_USERNAME/assemblage.git


ETAPE 5 : POUSSER VOTRE CODE SUR GITHUB
════════════════════════════════════════════════════════════════════════════

Exécutez dans votre terminal :

$ git remote add origin https://github.com/VOTRE_USERNAME/assemblage.git
$ git branch -M main
$ git push -u origin main

IMPORTANT: Remplacez "VOTRE_USERNAME" par votre vrai nom d'utilisateur GitHub !

Vous devriez voir :
✓ Enumerating objects: 150, done.
✓ ...
✓ Branch 'main' set up to track remote branch 'main' from 'origin'.


ETAPE 6 : DEPLOYER SUR RENDER
════════════════════════════════════════════════════════════════════════════

1. Allez sur https://render.com
2. Connectez-vous avec votre compte GitHub
3. Cliquez "New +" → "Web Service"
4. Sélectionnez votre dépôt "assemblage"
5. Remplissez les informations :
   - Name: assemblage-app
   - Environment: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
   - Plan: Free
6. Cliquez "Create Web Service"

Render va commencer le déploiement (2-3 minutes).


ETAPE 7 : CONFIGURER LES VARIABLES D'ENVIRONNEMENT
════════════════════════════════════════════════════════════════════════════

1. Dans le dashboard Render, allez à "Environment"
2. Ajoutez les variables :
   - FLASK_ENV = production
   - SECRET_KEY = your_random_secret_key_here
3. Cliquez "Save"

Render va redéployer automatiquement.


ETAPE 8 : TESTER VOTRE APPLICATION
════════════════════════════════════════════════════════════════════════════

1. Attendez 2-3 minutes
2. Allez sur https://assemblage-app.onrender.com
3. Vous devriez voir votre application !
4. Testez la connexion :
   Email: admin@assemblage.ch

════════════════════════════════════════════════════════════════════════════
