ETAPE 1: OUVRIR LE TERMINAL
════════════════════════════════════════════════════════════════════════════

Windows:
────────
1. Appuyez sur: Windows + R
2. Tapez: cmd
3. Appuyez sur: Entrée


Mac:
────
1. Appuyez sur: Cmd + Espace
2. Tapez: terminal
3. Appuyez sur: Entrée


Linux:
──────
1. Appuyez sur: Ctrl + Alt + T


ETAPE 2: ALLER DANS LE DOSSIER DE VOTRE APPLICATION
════════════════════════════════════════════════════════════════════════════

Tapez cette commande (remplacez le chemin par le vôtre):

Windows:
────────
cd C:\Users\Lenovo\Downloads\Technical Specification Sheet Review (4)

OU si votre dossier est ailleurs:
cd C:\chemin\vers\votre\dossier


Mac/Linux:
──────────
cd /Users/votre_nom/chemin/vers/votre/dossier


ETAPE 3: INITIALISER GIT
════════════════════════════════════════════════════════════════════════════

Tapez:
git init


ETAPE 4: AJOUTER VOS FICHIERS
════════════════════════════════════════════════════════════════════════════

Tapez:
git add .


ETAPE 5: CREER UN COMMIT
════════════════════════════════════════════════════════════════════════════

Tapez:
git commit -m "Déploiement automatisé de Assembl'âge"


ETAPE 6: RENOMMER LA BRANCHE
════════════════════════════════════════════════════════════════════════════

Tapez:
git branch -M main


ETAPE 7: AJOUTER LE REMOTE GITHUB
════════════════════════════════════════════════════════════════════════════

Tapez (remplacez VOTRE_USERNAME par votre nom d'utilisateur GitHub):

git remote add origin https://github.com/VOTRE_USERNAME/assemblage.git

Exemple:
git remote add origin https://github.com/jean123/assemblage.git


ETAPE 8: ENVOYER VOS FICHIERS VERS GITHUB
════════════════════════════════════════════════════════════════════════════

Tapez:
git push -u origin main

Vous devrez peut-être entrer votre nom d'utilisateur GitHub et mot de passe.

════════════════════════════════════════════════════════════════════════════

VERIFIER QUE LES FICHIERS SONT SUR GITHUB
════════════════════════════════════════════════════════════════════════════

2. Vous devriez voir tous vos fichiers

APRES AVOIR IMPORTE VOS FICHIERS
════════════════════════════════════════════════════════════════════════════
