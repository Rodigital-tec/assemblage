ETAPES DE DEPLOIEMENT
════════════════════════════════════════════════════════════════════════════

1️⃣  INITIALISER GIT
    $ git init
    $ git add .
    $ git commit -m "Préparation déploiement"

2️⃣  CREER UN DEPOT GITHUB
    → https://github.com/new
    → Créez un dépôt "assemblage"
    → Sélectionnez "Public"

3️⃣  POUSSER VOTRE CODE
    $ git remote add origin https://github.com/VOTRE_USERNAME/assemblage.git
    $ git branch -M main
    $ git push -u origin main

4️⃣  DEPLOYER SUR RENDER
    → https://render.com
    → Connectez-vous avec GitHub
    → Créez un "Web Service"
    → Sélectionnez votre dépôt

5️⃣  CONFIGURER LES VARIABLES
    → FLASK_ENV=production
    → SECRET_KEY=your_secret_key

6️⃣  ATTENDRE LE DEPLOIEMENT
    → 2-3 minutes
    → Votre app sera accessible à :
    → https://assemblage-app.onrender.com
