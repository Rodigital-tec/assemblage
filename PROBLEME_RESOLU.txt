POURQUOI ?
════════════════════════════════════════════════════════════════════════════

Le dépôt "assemblage" n'existe pas encore sur GitHub !

Le script vous demande votre nom d'utilisateur, mais il ne crée pas le dépôt.
Vous devez le créer manuellement d'abord.


SOLUTION
════════════════════════════════════════════════════════════════════════════

Suivez ces 3 étapes simples:

1️⃣  CREER LE DEPOT GITHUB
    └─ Allez sur: https://github.com/new
    └─ Créez un dépôt nommé "assemblage"
    └─ Cochez "Public"
    └─ Ne cochez PAS "Initialize repository"
    └─ Cliquez sur "Create repository"

2️⃣  LANCER LE SCRIPT
    └─ Windows: Double-cliquez sur deploy.bat
    └─ Mac/Linux: bash deploy.sh
    └─ Entrez votre nom d'utilisateur GitHub
    └─ Attendez que le script termine

3️⃣  DEPLOYER SUR RENDER
    └─ Allez sur: https://render.com
    └─ Connectez-vous avec GitHub
    └─ Cliquez sur "New +" puis "Web Service"
    └─ Vous verrez maintenant votre dépôt "assemblage"
    └─ Sélectionnez-le et configurez
