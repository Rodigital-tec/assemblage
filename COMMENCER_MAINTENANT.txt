BIENVENUE !
════════════════════════════════════════════════════════════════════════════

Votre application Assembl'âge est prête à être déployée sur Render.

Suivez les étapes ci-dessous pour commencer.


ETAPE 1 : LIRE LA DOCUMENTATION (5 minutes)
════════════════════════════════════════════════════════════════════════════

Ouvrez et lisez l'un de ces fichiers :

📖 LISEZMOI.txt              ← Commencez par celui-ci !
📖 QUICK_START.md            ← Guide rapide
📖 DEMARRER_ICI.txt          ← Guide complet


ETAPE 2 : CREER LES COMPTES (5 minutes)
════════════════════════════════════════════════════════════════════════════

1. Créez un compte GitHub
   → https://github.com
   → Cliquez "Sign up"

2. Créez un compte Render
   → https://render.com
   → Cliquez "Sign up"
   → Connectez-vous avec GitHub


ETAPE 3 : INITIALISER GIT (2 minutes)
════════════════════════════════════════════════════════════════════════════

Ouvrez un terminal/PowerShell dans votre dossier et exécutez :

$ git init
$ git add .
$ git commit -m "Préparation déploiement"


ETAPE 4 : CREER UN DEPOT GITHUB (5 minutes)
════════════════════════════════════════════════════════════════════════════

1. Allez sur https://github.com/new
2. Remplissez :
   - Repository name: assemblage
   - Description: Application Assembl'âge
   - Sélectionnez "Public"
3. Cliquez "Create repository"
4. Copiez l'URL du dépôt


ETAPE 5 : POUSSER VOTRE CODE (2 minutes)
════════════════════════════════════════════════════════════════════════════

Exécutez dans votre terminal :

$ git remote add origin https://github.com/VOTRE_USERNAME/assemblage.git
$ git branch -M main
$ git push -u origin main

IMPORTANT: Remplacez "VOTRE_USERNAME" par votre vrai nom d'utilisateur GitHub !


ETAPE 6 : DEPLOYER SUR RENDER (5 minutes)
════════════════════════════════════════════════════════════════════════════

1. Allez sur https://render.com
2. Connectez-vous avec votre compte GitHub
3. Cliquez "New +" → "Web Service"
4. Sélectionnez votre dépôt "assemblage"
5. Remplissez les informations :
   - Name: assemblage-app
   - Environment: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
   - Plan: Free
6. Cliquez "Create Web Service"


ETAPE 7 : CONFIGURER LES VARIABLES (2 minutes)
════════════════════════════════════════════════════════════════════════════

1. Dans le dashboard Render, allez à "Environment"
2. Ajoutez les variables :
   - FLASK_ENV = production
   - SECRET_KEY = your_random_secret_key_here
3. Cliquez "Save"


ETAPE 8 : ATTENDRE LE DEPLOIEMENT (3 minutes)
════════════════════════════════════════════════════════════════════════════

1. Attendez 2-3 minutes
2. Render va déployer votre application
3. Vous recevrez une URL : https://assemblage-app.onrender.com


ETAPE 9 : TESTER VOTRE APPLICATION (2 minutes)
════════════════════════════════════════════════════════════════════════════

1. Allez sur https://assemblage-app.onrender.com
2. Testez la connexion :
   Email: admin@assemblage.ch
   Mot de passe: admin123
3. Vérifiez que tout fonctionne


ETAPE 10 : PARTAGER VOTRE APPLICATION (1 minute)
════════════════════════════════════════════════════════════════════════════

Votre application est maintenant en ligne !

URL: https://assemblage-app.onrender.com

Vous pouvez :
✓ Partager l'URL avec d'autres
✓ Continuer à développer
✓ Pousser des mises à jour (redéploiement automatique)
