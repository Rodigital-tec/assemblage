→ https://github.com/new
→ Créez un dépôt "assemblage"

$ git push -u origin main

Étape 4: Déployer sur Render
→ https://render.com
→ Connectez-vous avec GitHub
→ Créez un "Web Service"
→ Sélectionnez votre dépôt

8. DEPANNAGE
════════════════════════════════════════════════════════════════════════════

PROBLEME: L'application ne démarre pas
SOLUTION:
1. Vérifiez les logs sur Render
2. Vérifiez que Procfile existe
3. Vérifiez que requirements.txt contient gunicorn
4. Vérifiez que app.py est à la racine

PROBLEME: Erreur 500
SOLUTION:
1. Vérifiez les variables d'environnement
2. Vérifiez que FLASK_ENV=production
3. Vérifiez que SECRET_KEY est défini
4. Consultez les logs sur Render

PROBLEME: Base de données non persistante
SOLUTION:
1. Utilisez PostgreSQL au lieu de SQLite
2. Configurez DATABASE_URL
3. Consultez DEPLOIEMENT_RENDER.md

PROBLEME: L'app s'arrête après 15 minutes
SOLUTION:
1. C'est normal sur le plan gratuit de Render
2. Passez à un plan payant
3. Ou utilisez un autre service
